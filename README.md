# cpsc4240-sysAdminSecurity

The wonderful file and directory permissions GUI, straight from your terminal!!
These are the dependencies
+ make
+ gcc
+ dialog
+ libncurses-dev

## Installing on Linux
To compile on linux make sure those packages are installed
In the directory run
~~~
make compile
~~~

## Installing on MacOS
To compile on MacOS brew install the packages as well
In the directory run
~~~
make mcompile
~~~
